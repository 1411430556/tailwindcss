## Tailwind + Framer Motion = Pocket

大约一周前，我们发布了一个全新的 Tailwind UI 模板，采用 Next.js 构建，并且（当然）使用了 Tailwind CSS。

我们称之为 <Link href="https://tailwindui.com/templates/pocket">Pocket</Link>，这是一个加载了大量有趣动画和交互的移动应用着陆页，由 <Link href="https://www.framer.com/motion/">Framer Motion</Link> 提供动力，基本上它是有史以来最酷的库。

<Link href="https://tailwindui.com/templates/pocket">
  <Image alt="了解 Pocket 模板" src={pocketPreview} />
</Link>

确保查看 <Link href="https://pocket.tailwindui.com/">在线预览</Link> 以获得完整体验——截图遗漏了所有最酷的部分，比如页面加载时绘制的图表、特性部分中的动画移动设备 UI，以及受到《青蛙过马路》启发的推荐动画。

关于这个模板有个有趣的故事——当我们最初设计和构建它时，实际上并没有计划在动画或交互方面做任何事情。我们在 Figma 中做了一个静态设计，大家都对此感到非常兴奋，然后按照计划大约三周半前构建了整个项目。

但一旦我们在浏览器中看到完成的模板，就觉得它显得有些僵硬。它在 Figma 中看起来很好，因为你会觉得一切都是静止的，但当我们在现实中看到它时，更像是一个截图，而不是一个真实的交互式网站。

看看 <Link href="https://tailwindui-template-wip-saas-02-7i25puvq2-tailwindlabs.vercel.app/">这个早期版本的模板</Link>，也许你会明白我的意思。

我们决定推迟发布几周，同时尝试一些可以为其注入活力的东西，经过尝试许多不同的想法，最终得到了现在的风格。

我对这个项目的成果感到非常激动，我认为这个代码库是一个非常有趣的案例研究，如果你想学习如何使用 Framer Motion 实现我们完成的一些酷炫效果。

和我们所有的模板一样，如果你拥有 <Link href="https://tailwindui.com/all-access">Tailwind UI 全访问</Link> 许可证，它与我们其他组件和模板一起包含在内，没有额外费用。

## Tailwind Jobs

我们已经讨论了大约两年的时间，考虑建立一个官方 Tailwind CSS 招聘平台，最后在过去几周决定尝试一下，看看会发生什么。

如果你是一家使用 Tailwind 的公司，正在寻找前端工程师，或者你是一名想在使用 Tailwind 的公司工作的开发者，<Link href="https://jobs.tailwindcss.com/">请查看 Tailwind Jobs</Link> 以了解更多信息。

<Link href="https://jobs.tailwindcss.com">
  <Image alt="Tailwind Jobs" src={tailwindJobs} />
</Link>

我们于 7 月 14 日开始代码库的建设，并于 8 月 2 日向公司推出了网站。我们采用 <Link href="https://laravel.com/">Laravel</Link>、<Link href="https://inertiajs.com/">Inertia</Link>、<Link href="https://reactjs.org/">React</Link>，当然还有 Tailwind CSS 来构建它。在仅仅 20 天内，利用这些工具构建出这样的东西，真是相当神奇！

我们仍在摸索定位以及如何让它成为使用 Tailwind 的公司寻找前端人才的最佳平台，所以如果你的公司有招聘工作，或者有任何反馈或想法，[给 Peter 发个邮件](mailto:peter@tailwindcss.com)——他在这个项目中担任主要负责人，并在寻找人们与之交谈。

我们正在尝试弄清楚的一件事是，如何使其清楚地说明，职位并不一定要严格是“整天用 Tailwind CSS 进行样式设计”才合适——实际上，世界上并没有很多这样的工作，甚至在真正 _制作_ Tailwind CSS 的公司中也没有这样的职位！

但是，很多人讨厌与 CSS 纠缠，尽管他们并不是每天都在编写它。

有一群开发者会很高兴地知道，尽管他们_确实_必须进行样式设计时，他们会用 Tailwind，而不是某种自定义 CSS 的混乱。

我们希望这个招聘平台能够成为寻找 React 开发者、Vue 开发者、Laravel 开发者、Rails 开发者等的地方——只要这个人可以在设计某些东西的时候使用 Tailwind，那就适合 Tailwind Jobs。
