大约八个月前我们 [开源了我们在 Tailwind CSS v4.0 上的进展](https://tailwindcss.zhcndoc.com/blog/tailwindcss-v4-alpha)。数百小时的修复 bug、令人筋疲力尽的向后兼容性工作，以及排查 Windows CI 故障后，我终于可以兴奋地标记第一个公开测试版发布。

正如我在发布第一个 alpha 版时所提到的，Tailwind CSS v4.0 是一个全新的引擎，旨在提升性能，并为现代网络设计。

- **为性能而构建** — 新引擎中的完整构建速度提高至 5 倍，增量构建速度超过 100 倍 — 以微秒为单位测量。
- **统一的工具链** — 内置的导入处理、厂商前缀和语法转换，无需额外的工具。
- **CSS 优先配置** — 重新构想的开发者体验，使您能够直接在 CSS 中自定义和扩展框架，而不是在 JavaScript 配置文件中。
- **为现代网络设计** — 基于原生级联层、广色域颜色，并包括对现代 CSS 特性的原生支持，如容器查询、`@starting-style`、弹出框等。

还有很多可以说的，但您可以在我们今天发布的新测试版文档中找到开始所需的一切：
