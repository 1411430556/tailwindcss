我们刚完成了一项巨大的品牌重塑项目，将之前被称为 Tailwind UI 的产品转变为 [Tailwind Plus](https://tailwindcss.com/plus)。

<Link href="https://tailwindcss.com/plus" className="my-12 block">
  <Image src={card} alt="Tailwind Plus" />
</Link>

在您做出任何结论之前，这里介绍一下我们并没有改变的内容：

- **无订阅定价** — 仍然是一口价购买，终身访问，与之前一样。
- **无升级费用** — 如果您已经拥有 Tailwind UI 的全访问许可，现在您也拥有了 Tailwind Plus 的许可。新客户的价格也没有变化。
- **所有相同的内容** — 您可以访问所有相同的内容，包括我们所有的组件、模板以及我们的 Catalyst UI 套件。

那么为什么会有这种改变呢？简单地说，我们希望为社区做许多事情，这些事情似乎不适合在 Tailwind UI 这个框架下进行。

以下是我们为 Tailwind Plus 探索的一些想法：

- **Tailwind Play 账户** — 这样您可以保存实验，轻松找到旧想法，甚至可以在不失去原始网址的情况下更新现有的 Play。
- **自定义编辑器主题** — 人们经常向我们询问我们使用的自定义语法主题，而 Tailwind Plus 让我们可以以一种有趣的方式向大家提供这些主题。
- **颜色调色板** — 针对 Tailwind CSS 默认颜色调色板的替代方案，由专家设计师为您精心策划。
- **排版主题** — 更多的排版风格，类似于我们排版插件所提供的，但提供不同的风格选择。
- **建立更强大的私密社区** — Tailwind 被数百万人使用，这太棒了，但也意味着公共渠道上有很多噪音。我们希望建立一个良好的社区，让您能与其他 UI 工程师就前端开发进行高质量对话，并直接与 Tailwind CSS 团队进行连接。
- **优先访问测试文档** — 在新特性正式发布之前，首先获取文档并帮助我们在稳定之前进行完善。
- **提前注册线下活动** — 例如我们主办的 Tailwind Connect 会议，在那里您可以与 Tailwind 团队互动并与社区的其他成员交流。

这些想法中有些进展得更快，有些可能并不是好的想法，但我真的很期待在未来几个月继续探索它们。

当然，我们也将继续投资于更多的 [组件示例](/plus/ui-blocks)、[模板](/plus/templates)以及 [Catalyst](/plus/ui-kit) 的改进！例如，我们迫不及待想要 **为 Catalyst 设计一个组合框**，而且我们也即将开始为不使用 React 的人原型制作 **简单的 HTML 版本模板**。

总之，Tailwind Plus 依然是您熟悉的高质量资源，但提供了全新的可能性和潜力。

做出这样的改变是一个很大的风险，但这是我多年来一直想要做的事情，我很兴奋终于能够启动并实施它。

祝贺一个新时代的到来 — 期待为您打造令人惊艳的新产品。
