## React + Vue libraries

除了直接从网站获取所需的图标外，您还可以安装我们的官方 React 和 Vue 库，以便快捷方便地将每个图标作为独立组件使用。

以下是 React 的示例：

```jsx
import { BeakerIcon } from "@heroicons/react/solid";

function MyComponent() {
  return (
    <div>
      <BeakerIcon className="h-5 w-5 text-blue-500" />
      <p>...</p>
    </div>
  );
}
```

<Link href="https://github.com/tailwindlabs/heroicons">查看 GitHub 上的文档</Link> 以了解更多信息。
