在六月份， [Brad Cornes](https://twitter.com/bradlc) 作为我们第一位团队成员加入了公司。那时我们没有博客来宣布此事，但迟到总比不到好，对吧？

您可能知道 Brad 是惊人的 [Tailwind CSS IntelliSense 插件](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) 的创造者，该插件专为 VS Code 开发，最初是在 2018 年发布的，并且至今安装次数超过 100,000 次！

<Image alt="Introducing linting for Tailwind CSS Intellisense" src={image} />

Brad 自 Tailwind 首次发布以来就一直在使用它，我在创建 Tailwind 社区的早期阶段认识了 Brad。令我立即感到印象深刻的是，他愿意解决极其困难的问题，能够提出创造性的解决方案，以及对深入前沿技术的热衷，并寻找有趣的应用案例。

当我们决定想要扩充团队时，第一时间想到的人就是 Brad，我在三月份就联系了他讨论这个想法。到现在他和我们已经合作了一个多月，和他一起工作非常棒，并从他的丰富经验和专业知识中受益匪浅。

Brad 一直在各个方面支持我们，参与了 IntelliSense 插件的开发、为 Tailwind UI 开发内部工具，甚至还建立了这个博客。他有一个惊人的能力，能够将一个复杂的难题，锁在一个房间里两个小时，然后带着优雅的解决方案出来。

他是团队的绝佳补充，我们对在未来几个月中期待的新事物感到非常兴奋，这些新事物的实现都离不开他的帮助。

请跟随 Brad [在 Twitter 上](https://twitter.com/bradlc) 关注他正在做的事情，以及一些令人兴奋的新项目的预告！
