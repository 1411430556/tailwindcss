## 设计工程师

我们正在招聘一名 [设计工程师](https://v3.tailwindcss.com/careers/design-engineer)，负责构建雄心勃勃的界面，原型化新想法，并推动 Tailwind CSS 的可能性边界。

<Link href="https://v3.tailwindcss.com/careers/design-engineer">
  <Image src={designEngineer} alt="设计工程师" />
</Link>

您的职责将包括：

- **为我们的开源项目、商业产品以及像 Tailwind Connect 的活动设计和构建雄心勃勃的营销网站**。
- **设计和原型化 Tailwind CSS 的新特性**，确保我们始终充分利用平台的潜力。
- **为 Tailwind UI 创建新的组件和模板**，将其从初步概念一直带到上线。
- **通过视觉演示增强我们的文档**，使人们能够轻松理解和应用复杂的 CSS 特性。
- **通过将您设计和构建的有趣事物分解为文章和社交媒体帖子来教导和激励我们的受众**。

以下是您在过去几个月中可能工作过的一些实际项目示例：

- **设计和构建 [Tailwind Connect](https://connect.tailwindcss.com/) 微网站** — 包括提出独特的基于 Markdown 的徽章设计体验。
- **为 Catalyst 制作 [第一个组件](https://catalyst.tailwindui.com/docs/dialog)** — 我们第一个完全组件化的 React UI 套件。
- **构建一个捕捉视频的工具，用于我们的 [Showcase](https://tailwindcss.zhcndoc.com/showcase) 网站** — 自动创建预览视频，让其感觉像一个真正的人在浏览网站。
- **构建 [交互演示](https://tailwindcss.zhcndoc.com/docs/height#dynamic-viewport-height) 以解释动态视口单位** — 随着 Tailwind CSS v3.4 的发布包含在文档中。

在您入职后，您将参与即将开展的项目，如：

- **为 Tailwind CSS v4.0 版本设计和构建一个交互式微网站**，其中包含精心制作的演示，以可视化方式传达最重要的改进。
- **原型化 Tailwind CSS 中基于滚动的动画 API**，在简便性和灵活性之间找到平衡，让我们的用户感觉自己拥有超能力。
- **探索带有自动深色模式支持的新配色方案**，仔细确定适用于不同项目的不同层次的正确抽象。
- **研究和设计 Tailwind CSS 的文本阴影支持**，终究实现这一目标。

这是一个完全远程的职位，薪水为 **275,000 美元**，面向东部（UTC-5）至中欧（UTC+1）时区的候选人开放。

## 职员软件工程师

我们正在招聘一名 [职员软件工程师](https://v3.tailwindcss.com/careers/staff-software-engineer)，负责开发 Tailwind CSS、Headless UI 及其支持的工具生态系统，如适用于 VS Code 的 Tailwind CSS IntelliSense 扩展。

<Link href="https://v3.tailwindcss.com/careers/staff-software-engineer">
  <Image src={staffEngineer} alt="职员工程师" />
</Link>

您的职责将包括：

- **为 Tailwind CSS 和 Headless UI 等项目构建和记录新功能**。
- **为上游项目贡献改进**，如 Lightning CSS、Next.js 和 Vite。
- **维护 CI 工作流程和自动化流程**，用于运行测试和构建发布。
- **管理来自社区的开放源代码库中的问题和贡献**。
- **定义和维护工程标准**，专注于代码质量、性能和文档，确保我们始终交付值得自豪的工作。
- **管理项目优先级并调整范围**，以在不牺牲顶层目标的情况下满足项目截止日期。

以下是您在过去几个月中可能工作过的一些实际项目示例：

- **创建一个用于提取工具类名称的 [Rust 库](https://github.com/tailwindlabs/tailwindcss/blob/0ee3508179966a767b7802a7262bb9d4ce2221da/oxide/crates/core/src/lib.rs)** — 一个能够尽快扫描整个模板文件目录并并行提取潜在类名的工具。
- **用 TypeScript 编写一个 [快速 CSS 解析器](https://github.com/tailwindlabs/tailwindcss/blob/0ee3508179966a767b7802a7262bb9d4ce2221da/packages/tailwindcss/src/css-parser.ts)** — 仅解析 Tailwind 内部所需的精确细节，寻找领域特定的快捷方式，使其比现有库更快以满足我们的需求。
- **修复 Headless UI 中的 [滚动锁定问题](https://github.com/tailwindlabs/headlessui/pull/2891)** — 确保对话框中的可滚动元素在 iOS 上仍然可以滚动。
- **为 Vite争取 [新 API](https://github.com/vitejs/vite/pull/16135)** — 为我们需要的改进准备一个有说服力的论据，以支持 Tailwind CSS，并与 Vite 核心团队协作原型解决方案。

在您入职后，您将参与即将开展的项目，如：

- **为 Next.js 开发一个 Tailwind CSS 捆绑插件** — 通过模块图生成您的 CSS，而不是扫描文件系统，积极与我们的 Vercel 联系人合作以确保正确。
- **为 Tailwind CSS 设计一组 IDE 集成 API** — 首级、稳定、文档齐全的 API，供 JetBrains 等编辑器团队依赖，以为他们的工具添加自动补全、lint 警告、类排序等功能。
- **为 Headless UI 设计、构建和记录一个 Tooltip 组件** — 确保其完全可访问，具有灵活但令人愉悦的 API。
- **为 Tailwind CSS v4.0 构建向后兼容层** — 重新引入对 JavaScript 配置文件的支持，探索 codemod 工具，并确保现有的社区插件尽可能与新引擎兼容。

这是一个完全远程的职位，薪水为 **275,000 美元**，面向东部（UTC-5）至中欧（UTC+1）时区的候选人开放。
