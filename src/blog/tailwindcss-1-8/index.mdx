Tailwind CSS v1.8 现在可用，提供了一系列新工具、几个新特性和一个令人兴奋的新实验！

## 新特性

- 新的 `font-variant-numeric` 工具 ([#2305](https://github.com/tailwindlabs/tailwindcss/pull/2305))
- 新的 `place-items`，`place-content`，`place-self`，`justify-items` 和 `justify-self` 工具 ([#2306](https://github.com/tailwindlabs/tailwindcss/pull/2306))
- 新的 `purge` 的 `preserveHtmlElements` 选项 ([#2283](https://github.com/tailwindlabs/tailwindcss/pull/2283))
- 新的 `purge` 的 `layers` 模式 ([#2288](https://github.com/tailwindlabs/tailwindcss/pull/2288))
- 支持将变体配置为函数 ([#2309](https://github.com/tailwindlabs/tailwindcss/pull/2309))
- 黑暗模式变体（实验性） ([#2279](https://github.com/tailwindlabs/tailwindcss/pull/2279))

## 变更

- 现在 `@layer` 中的 CSS 与相应的 `@tailwind` 规则进行了分组 ([#2312](https://github.com/tailwindlabs/tailwindcss/pull/2312))

## 弃用

- `conservative` purge 模式已被弃用，取而代之的是新的 `layers` 模式 ([#2288](https://github.com/tailwindlabs/tailwindcss/pull/2288))

查看 [GitHub 上的完整发布说明](https://github.com/tailwindlabs/tailwindcss/releases/tag/v1.8.0) 获取更多细节。
