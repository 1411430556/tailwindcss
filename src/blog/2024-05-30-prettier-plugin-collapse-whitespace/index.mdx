我们刚刚发布了 `prettier-plugin-tailwindcss` 的新版本，增加了在排序时去除不必要的空白和重复类的支持。

---

## 清理不必要的空白

当你复制类名或从列表中删除一个类名时，很容易留下需要清理的一些多余空白。

现在我们的 Prettier 插件将为你自动处理这些问题，因此你无需自己进行清理。

<Video src="https://assets.tailwindcss.com/blog/2024-05-30-prettier-plugin-collapse-whitespace/collapse-whitespace.mp4" />

---

## 移除重复的类

我们的 [VS Code 扩展](/docs/editor-setup#intellisense-for-vs-code) 很久以来一直在警告你关于重复类名的问题，但现在我们的 Prettier 插件可以自动为你移除这些重复的类。

<Video src="https://assets.tailwindcss.com/blog/2024-05-30-prettier-plugin-collapse-whitespace/remove-duplicates.mp4" />

---

要在自己的项目中体验这些改进，只需安装最新版本：

```sh
npm i prettier-plugin-tailwindcss@latest
```

如果你遇到任何问题，请在 [GitHub](https://github.com/tailwindlabs/prettier-plugin-tailwindcss/) 上告诉我们！
