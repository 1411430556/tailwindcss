自 Tailwind CSS v2.0 以来的第一次新功能更新现在来了，装满了许多令人兴奋的东西！我们将新的 JIT 引擎合并到核心，增加了对可组合 CSS 滤镜的一流支持，添加了混合模式工具，还有更多。

以下是一些亮点：

- [核心中的 JIT 引擎](#jit-engine-in-core)
- [可组合 CSS 滤镜 API](#new-filter-and-backdrop-filter-utilities)
- [新的混合模式工具](#new-blending-mode-utilities)
- [新的隔离工具](#new-isolation-utilities)

有关完整细节，请 <Link href="https://github.com/tailwindlabs/tailwindcss/releases/tag/v2.1.0">查看 GitHub 上的发布说明</Link>。

<a name="jit-engine-in-core"></a>

## 核心中的 JIT 引擎

我们在三月发布的 <Link href="https://blog.tailwindcss.com/just-in-time-the-next-generation-of-tailwind-css">全新 JIT 引擎</Link> 现在已经合并到核心，并可以通过在 `tailwind.config.js` 文件中使用新的 `mode` 选项来选择性启用：

```js
module.exports = {
  mode: "jit",
  purge: [
    // ...
  ],
  // ...
};
```

**此功能仍处于预览阶段**，这意味着一些细节可能会在我们解决问题时有所变化，并且不受语义版本控制的约束。

如果您之前使用过 `@tailwindcss/jit`，现在可以迁移到 Tailwind CSS v2.1，因为所有新的引擎开发将会在这里进行。

阅读 <Link href="https://v2.tailwindcss.com/docs/just-in-time-mode">即时模式文档</Link> 以了解更多信息。

<a name="new-filter-and-backdrop-filter-utilities"></a>

## 可组合 CSS 滤镜 API

这是一个重要更新——我们终于添加了一流的 CSS 滤镜支持！

它们的工作方式与我们的 `transform` 工具非常相似，您可以使用 `filter` 启用滤镜，并将其与像 `grayscale`、`blur-lg` 或 `saturate-200` 这样的工具组合，以便即时组合滤镜。

```html
<div class="blur-md grayscale invert filter ...">
  <!-- ... -->
</div>
```

……下面是 `backdrop-filter` 的使用方式：

```html
<div class="backdrop-blur backdrop-brightness-50 backdrop-filter ...">
  <!-- ... -->
</div>
```

查看 <Link href="https://v2.tailwindcss.com/docs/filter">滤镜</Link> 和 <Link href="https://v2.tailwindcss.com/docs/backdrop-filter">背景滤镜</Link> 以了解更多信息。我们很快会添加许多有用的视觉示例！

<a name="new-blending-mode-utilities"></a>

## 新的混合模式工具

我们添加了新的 `mix-blend-mode` 和 `background-blend-mode` 工具：

```html
<div class="mix-blend-multiply ...">
  <!-- ... -->
</div>
```

查看 <Link href="https://v2.tailwindcss.com/docs/mix-blend-mode">文档</Link> 以了解更多信息。

<a name="new-isolation-utilities"></a>

## 新的隔离工具

我们添加了新的 `isolate` 和 `isolation-auto` 工具用于处理 `isolation` 属性：

```html
<div class="isolate ...">
  <!-- ... -->
</div>
```

这对于作用域混合模式特性或 z-index 调整非常有帮助，并且功能强大。

查看 <Link href="https://v2.tailwindcss.com/docs/isolation">文档</Link> 以了解更多信息。

我还强烈推荐 <Link href="https://www.joshwcomeau.com/css/stacking-contexts/#airtight-abstractions-with-isolation">Josh Comeau 的这篇精彩文章</Link> 以查看其实际应用。

## 升级

Tailwind CSS v2.1 是一次增量升级，没有重大变更，因此要升级，您只需运行：

```bash
npm install tailwindcss@latest
```

如果您之前使用的是 `@tailwindcss/jit`，现在可以切换回 `tailwindcss` 并相应地更新您的 PostCSS 配置。

准备好升级了吗？ <Link href="https://npmjs.com/tailwindcss">从 npm 获取 →</Link>

此迁移确保内容符合新的 v4 格式，利用指定的组件和代码作为示例、摘要和代码块。
