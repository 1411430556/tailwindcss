## 示例

### 左对齐文本

使用 `text-left` 工具类使元素的文本左对齐：

<Figure>

<Example padding={false}>
  {
    <div className="px-4 sm:px-0">
      <div className="mx-auto max-w-lg p-8 text-sm/6 text-gray-900 dark:text-gray-200">
        <p className="text-left">
          所以我开始走入水中。我不会对你们撒谎，伙计们，我感到十分害怕。但我坚持下来了，当我越过波浪时，一种奇怪的平静笼罩了我。我不知道那是神灵的干预，还是所有生物之间的亲密关系，但我告诉你，杰里，那一刻，我 <em>是</em> 一名海洋生物学家。
        </p>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-left] -->
<p class="text-left">所以我开始走入水中...</p>
```

</Figure>
