## 示例

### 基本示例

使用 `break-inside-column` 和 `break-inside-avoid-page` 等工具来控制元素内列或页面的换行行为：

```html
<!-- [!code classes:break-inside-avoid-column] -->
<div class="columns-2">
  <p>那么，告诉你一件事，...</p>
  <p class="break-inside-avoid-column">当然，继续笑吧...</p>
  <p>也许我们可以不需要...</p>
  <p>看看。如果你认为这是...</p>
</div>
```
