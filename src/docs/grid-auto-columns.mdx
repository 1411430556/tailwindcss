## 示例

### 基础示例

使用像 `auto-cols-min` 和 `auto-cols-max` 这样的工具来控制隐式创建的网格列的大小：

```html
<!-- [!code classes:auto-cols-max] -->
<div class="grid auto-cols-max grid-flow-col">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```
