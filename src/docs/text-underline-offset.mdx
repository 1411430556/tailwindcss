## 示例

### 基本示例

使用 `underline-offset-<number>` 工具类，如 `underline-offset-2` 和 `underline-offset-4` 来改变文本下划线的偏移量：

<Figure>

<Example>
  {
    <div className="flex flex-col gap-8 text-gray-900 dark:text-gray-200">
      <div>
        <div className="mb-3 font-mono text-xs font-medium text-gray-500 dark:text-gray-400">underline-offset-1</div>
        <p className="text-lg font-medium underline underline-offset-1">快的棕色狐狸跳过懒狗。</p>
      </div>
      <div>
        <div className="mb-3 font-mono text-xs font-medium text-gray-500 dark:text-gray-400">underline-offset-2</div>
        <p className="text-lg font-medium underline underline-offset-2">快的棕色狐狸跳过懒狗。</p>
      </div>
      <div>
        <div className="mb-3 font-mono text-xs font-medium text-gray-500 dark:text-gray-400">underline-offset-4</div>
        <p className="text-lg font-medium underline underline-offset-4">快的棕色狐狸跳过懒狗。</p>
      </div>
      <div>
        <div className="mb-3 font-mono text-xs font-medium text-gray-500 dark:text-gray-400">underline-offset-8</div>
        <p className="text-lg font-medium underline underline-offset-8">快的棕色狐狸跳过懒狗。</p>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:underline-offset-1] -->
<p class="underline underline-offset-1">快的棕色狐狸...</p>
<!-- [!code classes:underline-offset-2] -->
<p class="underline underline-offset-2">快的棕色狐狸...</p>
<!-- [!code classes:underline-offset-4] -->
<p class="underline underline-offset-4">快的棕色狐狸...</p>
<!-- [!code classes:underline-offset-8] -->
<p class="underline underline-offset-8">快的棕色狐狸...</p>
```

</Figure>
