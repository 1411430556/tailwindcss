### 使用断点容器

使用 `container` 工具类将元素的最大宽度设置为当前断点的 `min-width`。如果你更喜欢针对固定的屏幕尺寸进行设计，而不是尝试适应完全流体的视口，这非常有用：

```html
<!-- [!code classes:container] -->
<div class="container">
  <!-- ... -->
</div>
```

注意，与您可能在其他框架中使用的容器不同，Tailwind 的容器不会自动居中，并且没有任何内置的水平填充。使用 `mx-auto` 和 `px-<number>` 工具类来添加这些：

```html
<!-- [!code classes:mx-auto,px-4] -->
<div class="container mx-auto px-4">
  <!-- ... -->
</div>
```
