每个字体大小工具类在没有提供行高时也会设置一个默认的行高。有关这些值及如何自定义它们的更多信息，请查看 [字体大小文档](/docs/font-size)。

### 移除行高

使用 `leading-none` 工具类将元素的行高设置为与其字体大小相等：

<Figure>

<Example>
  {
    <p className="mx-auto max-w-2xs text-2xl leading-none text-gray-900 dark:text-gray-200">
      机敏的棕色狐狸跳过懒狗。
    </p>
  }
</Example>

```html
<!-- [!code classes:leading-none] -->
<p class="text-2xl leading-none ...">机敏的棕色狐狸...</p>
```

</Figure>
