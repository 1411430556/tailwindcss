## 示例

### 截断文本

使用 `truncate` 工具类以防止文本换行，并在需要时用省略号 (…) 来截断溢出的文本：

<Figure>

<Example padding={false}>
  {
    <p className="mx-auto max-w-xs truncate border-x border-x-pink-400/30 py-8 text-gray-900 dark:text-gray-200">
      英语主要词典中最长的单词是{" "}
      <span className="font-bold">pneumonoultramicroscopicsilicovolcanoconiosis,</span> 这个词指的是一种由于
      吸入非常细的硅酸盐颗粒（特别是来自火山）而引发的肺部疾病；在医学上，它与硅肺病是相同的。
    </p>
  }
</Example>

```html
<!-- [!code classes:truncate] -->
<p class="truncate">英语主要词典中最长的单词是...</p>
```

</Figure>

      <span className="font-bold">pneumonoultramicroscopicsilicovolcanoconiosis,</span> 这个词指的是一种由于
      吸入非常细的硅酸盐颗粒（特别是来自火山）而引发的肺部疾病；在医学上，它与硅肺病是相同的。
    </p>
  }
</Example>

```html
<!-- [!code classes:text-ellipsis] -->
<p class="overflow-hidden text-ellipsis">英语主要词典中最长的单词是...</p>
```

</Figure>

### 裁剪文本

使用 `text-clip` 工具类在内容区域的边界处截断文本：

<Figure>

<Example padding={false}>
  {
    <p className="mx-auto max-w-xs overflow-hidden border-x border-x-pink-400/30 py-8 text-clip text-gray-900 dark:text-gray-200">
      英语主要词典中最长的单词是{" "}
      <span className="font-bold">pneumonoultramicroscopicsilicovolcanoconiosis,</span> 这个词指的是一种由于
      吸入非常细的硅酸盐颗粒（特别是来自火山）而引发的肺部疾病；在医学上，它与硅肺病是相同的。
    </p>
  }
</Example>

```html
<!-- [!code classes:text-clip] -->
<p class="overflow-hidden text-clip">英语主要词典中最长的单词是...</p>
```

</Figure>

这是浏览器的默认行为。
