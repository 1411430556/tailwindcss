### 基本示例

使用 `isolate` 和 `isolation-auto` 工具来控制元素是否应该显式创建新的堆叠上下文：

```html
<!-- [!code classes:isolate] -->
<div class="isolate ...">
  <!-- ... -->
</div>
```
