### 使用负值

使用负值在 Tailwind 默认的命名字母间距缩放中并没有太大意义，但如果你自定义了你的缩放以使用数字，则可以非常有用：

```css
@theme {
  --tracking-1: 0em;
  --tracking-2: 0.025em;
  --tracking-3: 0.05em;
  --tracking-4: 0.1em;
}
```

要使用负的字母间距值，请在类名之前加上一个短划线，将其转换为负值：

```html
<!-- [!code classes:-tracking-2] -->
<p class="-tracking-2">敏捷的棕色狐狸 ...</p>
```
