## 示例

### 基础示例

使用 `break-before-column` 和 `break-before-page` 等工具来控制在元素之前列或页面换行的行为：

```html
<!-- [!code classes:break-before-column] -->
<div class="columns-2">
  <p>好吧，让我告诉你一些事情，...</p>
  <p class="break-before-column">当然，可以笑啊...</p>
  <p>也许我们可以不需要...</p>
  <p>听着。如果你认为这是...</p>
</div>
```
