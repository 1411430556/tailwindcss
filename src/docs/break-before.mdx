## 示例

### 基本示例

使用 `break-before-column` 和 `break-before-page` 等实用工具来控制在元素前如何进行列或页面断点：

```html
<!-- [!代码 类:break-before-column] -->
<div class="columns-2">
  <p>好吧，让我告诉你一些事情，...</p>
  <p class="break-before-column">当然，继续，笑吧...</p>
  <p>也许我们可以没有...</p>
  <p>看。如果你认为这是...</p>
</div>
```
