## 示例

### 基本示例

使用 `content-[<value>]` 语法，以及 `before` 和 `after` 变体，设置 `::before` 和 `::after` 伪元素的内容：

<Figure>

<Example>
  {
    <div className="mx-auto w-full max-w-md text-gray-500 dark:text-gray-400">
      更高的分辨率意味着的不仅仅是图像质量的提升。使用 Retina 6K 显示器，{" "}
      <a
        href="https://www.apple.com/pro-display-xdr/"
        className="font-medium text-blue-600 after:text-sm after:font-bold after:content-['_↗'] dark:text-sky-400"
        target="_blank"
      >
        Pro Display XDR
      </a>{" "}
      为您提供了比 5K 显示器多近 40% 的屏幕空间。
    </div>
  }
</Example>

```html
<!-- [!code classes:after:content-['_↗']] -->
<!-- prettier-ignore -->
<p>更高的分辨率意味着的不仅仅是图像质量的提升。使用
Retina 6K 显示器，<a class="text-blue-600 after:content-['_↗']" href="...">
Pro Display XDR</a> 为您提供了比 5K 显示器多近 40% 的屏幕空间。</p>
```

</Figure>

### 引用属性值

使用 `content-[attr(<name>)]` 语法来引用使用 `attr()` CSS 函数存储在属性中的值：

<Figure>

<Example>
  {
    <p
      before="你好，世界"
      className="text-center font-semibold text-gray-900 before:content-[attr(before)] dark:text-gray-200"
    />
  }
</Example>

```html
<!-- [!code classes:before:content-[attr(before)]] -->
<p before="你好，世界" class="before:content-[attr(before)] ...">
  <!-- ... -->
</p>
```

</Figure>

### 使用空格和下划线

由于空格在 HTML 中表示类的结束，因此将任意值中的任何空格替换为下划线：

<Figure>

<Example>
  {<p className="text-center font-semibold text-gray-900 before:content-['你好_世界'] dark:text-gray-200" />}
</Example>

```html
<!-- [!code classes:before:content-['你好_世界']] -->
<p class="before:content-['你好_世界'] ...">
  <!-- ... -->
</p>
```

</Figure>

如果您需要包含实际的下划线，可以通过用反斜杠转义来实现这一点：

<Figure>

<Example>
  {<p className="text-center font-semibold text-gray-900 before:content-['你好\_世界'] dark:text-gray-200" />}
</Example>

```html
<!-- [!code classes:before:content-['你好\_世界']] -->
<p class="before:content-['你好\_世界']">
  <!-- ... -->
</p>
```

</Figure>

### 使用 CSS 变量

使用 <code>content-{'(<custom-property>)'}</code> 语法，通过 CSS 变量控制 `::before` 和 `::after` 伪元素的内容：

```html
<!-- [!code classes:content-(--my-content)] -->
<p class="content-(--my-content)">
  <!-- ... -->
</p>
```

这只是 <code>content-{'[var(<custom-property>)]'}</code> 的缩写，自动为您添加 `var()` 函数。
