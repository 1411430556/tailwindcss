## 示例

### 基本示例

使用诸如 `auto-rows-min` 和 `auto-rows-max` 的工具类来控制隐式创建的网格行的大小：

```html
<!-- [!code classes:auto-rows-max] -->
<div class="grid grid-flow-row auto-rows-max">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```
