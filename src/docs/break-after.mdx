## 示例

### 基本示例

使用 `break-after-column` 和 `break-after-page` 等工具类来控制元素之后的列或页面的换行行为：

```html
<!-- [!code classes:break-after-column] -->
<div class="columns-2">
  <p>好吧，让我告诉你一些事情，...</p>
  <p class="break-after-column">当然，继续，笑吧...</p>
  <p>也许我们可以没有...</p>
  <p>看。如果你觉得这是...</p>
</div>
```
