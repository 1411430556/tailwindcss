### 移除滤镜

使用 `filter-none` 工具来移除应用于元素的所有滤镜：

```html
<!-- [!code classes:md:filter-none] -->
<img class="blur-md brightness-150 invert md:filter-none" src="/img/mountains.jpg" />
```
