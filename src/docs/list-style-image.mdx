### 使用 CSS 变量

使用 <code>list-image-{'(<custom-property>)'}</code> 语法通过 CSS 变量来控制列表项的标记图像：

```html
<!-- [!code classes:list-image-(--my-list-image)] -->
<ul class="list-image-(--my-list-image)">
  <!-- ... -->
</ul>
```

这只是 <code>list-image-{'[var(<custom-property>)]'}</code> 的简写形式，为您自动添加 `var()` 函数。

### 移除标记图像

使用 `list-image-none` 工具类从列表项中移除现有标记图像：

```html
<!-- [!code classes:list-image-none] -->
<ul class="list-image-none">
  <!-- ... -->
</ul>
```
