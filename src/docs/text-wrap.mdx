## 示例

### 允许文本换行

使用 `text-wrap` 工具在文本中逻辑点将溢出的文本换到多行：

<Figure>

<Example padding={false}>
  {
    <div className="mx-auto grid max-w-xs gap-4 border-x border-x-pink-400/30 py-8 text-gray-700 dark:text-gray-400">
      <div className="text-xl font-semibold text-gray-900 dark:text-white">心爱的曼哈顿汤摊关闭</div>
      <p className="text-sm/6">
        纽约人在这个冬天感到寒意更重，因为这座城市最受尊敬的汤摊意外关闭，接连发生的事件让社区感到困惑。
      </p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-wrap] -->
<article class="text-wrap">
  <h3>心爱的曼哈顿汤摊关闭</h3>
  <p>纽约人在这个冬天感到寒意更重...</p>
</article>
```

</Figure>

          心爱的曼哈顿汤摊关闭

        </p>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-nowrap] -->
<article class="text-nowrap">
  <h3>心爱的曼哈顿汤摊关闭</h3>
  <p>纽约人在这个冬天感到寒意更重...</p>
</article>
```

</Figure>

### 平衡文本换行

使用 `text-balance` 工具将文本均匀分布在每行上：

<Figure>

<Example padding={false}>
  {
    <div className="mx-auto grid max-w-xs gap-4 border-x border-x-pink-400/30 py-8 text-gray-700 dark:text-gray-400">
      <div className="text-xl font-semibold text-balance text-gray-900 dark:text-white">
        心爱的曼哈顿汤摊关闭
      </div>
      <p className="text-sm/6">
        纽约人在这个冬天感到寒意更重，因为这座城市最受尊敬的汤摊意外关闭，接连发生的事件让社区感到困惑。
      </p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-balance] -->
<article>
  <h3 class="text-balance">心爱的曼哈顿汤摊关闭</h3>
  <p>纽约人在这个冬天感到寒意更重...</p>
</article>
```

</Figure>

出于性能原因，浏览器将文本平衡限制在大约 6 行或更少的块上，最适合用于标题。
