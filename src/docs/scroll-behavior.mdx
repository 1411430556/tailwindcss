## 示例

### 使用平滑滚动

使用 `scroll-smooth` 工具类在元素内启用平滑滚动：

```html
<!-- [!code classes:scroll-smooth] -->
<html class="scroll-smooth">
  <!-- ... -->
</html>
```

设置 `scroll-behavior` 仅会影响浏览器触发的滚动事件。

### 使用正常滚动

使用 `scroll-auto` 工具类恢复为浏览器的默认滚动行为：

```html
<!-- [!code classes:scroll-auto] -->
<html class="scroll-smooth md:scroll-auto">
  <!-- ... -->
</html>
```
