## 示例

### 正常

使用 `whitespace-normal` 工具类使文本在元素内正常换行。换行符和空格将被折叠：

<Figure>

<Example padding={false}>
  {
    <div className="mx-auto max-w-sm border-x border-x-pink-400/30 py-8 text-gray-900 dark:text-gray-200">
      <p
        className="whitespace-normal"
        children={
          "大家好！\n\n快到2022年了，我们仍然不知道是否有外星人生活在我们当中，还是我们不知道？也许写这篇文章的人就是外星人。\n\n你永远不会知道。"
        }
      />
    </div>
  }
</Example>

```html
<!-- [!code classes:whitespace-normal] -->
<!-- prettier-ignore -->
<p class="whitespace-normal">大家好！

快到2022年了，我们仍然不知道是否有外星人生活在我们当中，还是我们不知道？也许写这篇文章的人就是外星人。

你永远不会知道。</p>
```

</Figure>

        }
      />
    </div>
  }
</Example>

```html

你永远不会知道。</p>
```

</Figure>
