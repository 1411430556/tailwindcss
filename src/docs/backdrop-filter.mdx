### 移除滤镜

使用 `backdrop-filter-none` 工具类移除应用于元素的所有背景滤镜：

```html
<!-- [!code classes:md:backdrop-filter-none] -->
<div class="backdrop-blur-md backdrop-brightness-150 md:backdrop-filter-none"></div>
```
