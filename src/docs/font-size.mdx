## 自定义主题

<CustomizingYourTheme utility="text" name="字体大小" customValue="0.625rem" customName="tiny">

您还可以为字体大小提供默认的 `line-height`、`letter-spacing` 和 `font-weight` 值：

```css
@theme {
  --text-tiny: 0.625rem;
  --text-tiny--line-height: 1.5rem; /* [!code highlight] */
  --text-tiny--letter-spacing: 0.125rem; /* [!code highlight] */
  --text-tiny--font-weight: 500; /* [!code highlight] */
}
```

</CustomizingYourTheme>
