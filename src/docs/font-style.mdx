## 示例

### 斜体文本

使用 `italic` 工具类将文本设置为斜体：

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 italic dark:text-gray-200">
      快速的棕色狐狸跳过懒惰的狗。
    </p>
  }
</Example>

```html
<!-- [!code classes:italic] -->
<p class="italic ...">快速的棕色狐狸 ...</p>
```

</Figure>

### 正常显示文本

使用 `not-italic` 工具类将文本正常显示：

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 not-italic dark:text-gray-200">
      快速的棕色狐狸跳过懒惰的狗。
    </p>
  }
</Example>

```html
<!-- [!code classes:not-italic] -->
<p class="not-italic ...">快速的棕色狐狸 ...</p>
```

</Figure>
