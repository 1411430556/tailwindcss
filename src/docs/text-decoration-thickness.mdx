## 示例

### 基本示例

使用 `decoration-<number>` 工具类，如 `decoration-2` 和 `decoration-4` 来改变元素的 [文本装饰](/docs/text-decoration-line) 厚度：

<Figure>

<Example>
  {
    <div className="flex flex-col gap-8 text-gray-900 dark:text-gray-200">
      <div>
        <div className="mb-3 font-mono text-sm font-medium text-gray-500 dark:text-gray-400">decoration-1</div>
        <p className="text-lg font-medium underline decoration-1">快棕色狐狸跃过懒狗。</p>
      </div>
      <div>
        <div className="mb-3 font-mono text-sm font-medium text-gray-500 dark:text-gray-400">decoration-2</div>
        <p className="text-lg font-medium underline decoration-2">快棕色狐狸跃过懒狗。</p>
      </div>
      <div>
        <div className="mb-3 font-mono text-sm font-medium text-gray-500 dark:text-gray-400">decoration-4</div>
        <p className="text-lg font-medium underline decoration-4">快棕色狐狸跃过懒狗。</p>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:decoration-1] -->
<p class="underline decoration-1">快棕色狐狸...</p>
<!-- [!code classes:decoration-2] -->
<p class="underline decoration-2">快棕色狐狸...</p>
<!-- [!code classes:decoration-4] -->
<p class="underline decoration-4">快棕色狐狸...</p>
```

</Figure>
