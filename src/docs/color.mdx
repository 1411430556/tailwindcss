## 示例

### 基本示例

使用类似 `text-blue-600` 和 `text-sky-400` 的工具类来控制元素的文本颜色：

<Figure>

<Example>
  {
    <div className="relative text-center text-xl leading-6 font-medium">
      <p className="text-blue-600 dark:text-sky-400">敏捷的棕色狐狸跳过懒狗。</p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-blue-600,dark:text-sky-400] -->
<p class="text-blue-600 dark:text-sky-400">敏捷的棕色狐狸...</p>
```

</Figure>

### 更改不透明度

使用颜色不透明度修饰符来控制元素文本颜色的不透明度：

<Figure>

<Example>
  {
    <div className="space-y-4 text-center text-xl leading-6 font-medium">
      <p className="text-blue-600/100 dark:text-sky-400/100">敏捷的棕色狐狸跳过懒狗。</p>
      <p className="text-blue-600/75 dark:text-sky-400/75">敏捷的棕色狐狸跳过懒狗。</p>
      <p className="text-blue-600/50 dark:text-sky-400/50">敏捷的棕色狐狸跳过懒狗。</p>
      <p className="text-blue-600/25 dark:text-sky-400/25">敏捷的棕色狐狸跳过懒狗。</p>
    </div>
  }
</Example>

```html
<!-- [!code word:\/100] -->
<!-- [!code word:\/75] -->
<!-- [!code word:\/50] -->
<!-- [!code word:\/25] -->
<p class="text-blue-600/100 dark:text-sky-400/100">敏捷的棕色狐狸...</p>
<p class="text-blue-600/75 dark:text-sky-400/75">敏捷的棕色狐狸...</p>
<p class="text-blue-600/50 dark:text-sky-400/50">敏捷的棕色狐狸...</p>
<p class="text-blue-600/25 dark:text-sky-400/25">敏捷的棕色狐狸...</p>
```

</Figure>
